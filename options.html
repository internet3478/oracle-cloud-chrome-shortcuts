Here you can configure your order and preferred shortcuts for Oracle Cloud Applications (ERP). You can select between two types of shortcuts: predefined and customs.
- Predefined Shortcuts are shortcuts created for a specific use.
1. Person Management: Go to Person Management search page. Optionally input a person number (only numbers) or a name (and press 'Enter' or click on button).
2. Extract Definitions: Go to Extract Definitions search page.
3. Checklist: Go to Checklist search page
4. Assignment: Input a person number and an effective date (and press 'Enter' or click on button) to get the assignment page only. This require a credential that can invoke REST API.
5. Security Console: Users: Optionally input a name and press 'Enter' or click on button to go to Security Console Users search page.
6. User Entities: Go to User Entities search page.
7. View Person: Input a person number (and press 'Enter' or click on button) and it will display in a Modial Dialog box the Person Details. This require a credential that can invoke REST API.

- Custom Shortcuts are shortcuts that you can create. Assign a name to the button, and the links.
1. Custom Navigator: It will click on the Navigator and the click on the "defined" links starting from the menu.
Format: Custom Navigator | Button Title | Link 1
Example 1: Custom Navigator | ED | Data Exchange
2. Custom Global Search: It will load setup and maintainance page, open the Taskpane, click on Search, input and search the "defined" text, and then click on the link.
Format: Custom Global Search | Button Title | Link 1 | Link 2
Example 1: Custom Global Search | Go To FF | Fast Formulas
3. Custom Quick Actions: It will go to the Home screen where the Quick Actions are, and click on the "defined" text
Format: Custom Quick Actions | Button Title | Link 1
Example 1: Custom Quick Actions | Go to Contracts | Contracts
